            IT Project Cost
Веб-приложение для управления IT-проектами с расчетом стоимости, себестоимости и прибыли.

        📋 О проекте
IT Project Cost — это полноценный инструмент для автоматизации учета IT-проектов, позволяющий:

📁 Управлять проектами — создание, просмотр, удаление проектов

💰 Рассчитывать финансы — автоматический расчет себестоимости, прибыли и рентабельности

👨‍💻 Вести базу сотрудников — учет сотрудников с зарплатами и налогами

💻 Управлять оборудованием — учет оборудования и его стоимости

🏢 Работать с клиентами — ведение клиентской базы

👥 Создавать команды — объединение пользователей в команды

💬 Общаться в чате — командный чат для обсуждения проектов

🔐 Безопасный вход — двухфакторная аутентификация через Telegram

        🚀 Демо
После запуска приложение доступно по адресу: http://localhost:8001

⚙️ Установка и запуск
Требования
Python 3.8 или выше

pip (менеджер пакетов Python)

Telegram аккаунт (для двухфакторной аутентификации)

            Шаги установки
    1. Клонировать репозиторий

bash
git clone https://github.com/yourusername/it-project-cost.git
cd it-project-cost

    2. Создать виртуальное окружение

bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate

    3. Установить зависимости

bash
pip install -r requirements.txt
Или установить вручную:

bash
pip install fastapi uvicorn sqlalchemy passlib requests python-dotenv pydantic-settings

    4. Настроить переменные окружения

Создайте файл .env в корне проекта:

env
DATABASE_URL=sqlite:///./it_project.db
TELEGRAM_BOT_TOKEN=ваш_токен_бота

    5. Настройка Telegram бота

Напишите @BotFather в Telegram

Используйте команду /newbot

Придумайте имя и username для бота

Получите токен и добавьте его в .env

    6. Запустить приложение

bash
python main.py
Сервер запустится по адресу: http://localhost:8001

📊 Финансовые расчеты
Формулы расчета
1. Себестоимость ресурса

text
cost_price = hours_days × 1000
2. Стоимость ресурса с маржинальностью

text
total_cost = cost_price × (1 + margin_percent / 100)
3. Финансовые показатели проекта

text
Общая стоимость проекта = sum(всех ресурсов.total_cost)
Себестоимость проекта = sum(всех ресурсов.cost_price)
Чистая прибыль = Стоимость для заказчика - Себестоимость
Рентабельность = (Прибыль / Себестоимость) × 100%
Пример расчета
Параметр	Значение
Количество дней	100
Себестоимость	100,000 руб
Маржинальность	30%
Стоимость с маржой	130,000 руб
Налог	20%
Стоимость для клиента	156,000 руб
Чистая прибыль	56,000 руб

🔐 Двухфакторная аутентификация
Процесс входа в систему:

Ввод логина и пароля — проверка учетных данных

Генерация кода — создается 6-значный код

Отправка в Telegram — код приходит в личные сообщения

Ввод кода — пользователь вводит полученный код

Успешный вход — создается сессионный токен

Важно: Код действителен 5 минут и может быть использован только один раз.
